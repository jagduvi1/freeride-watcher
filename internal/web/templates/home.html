How it works

1. Create a watch

Specify origin & destination city, optional time window, and whether you want a one-time or repeating alert.

2. We poll the API

Routes are fetched every few minutes from the official Hertz Freerider API with jitter to be a polite client.

3. Get notified

When a matching route appears, you receive a push notification (or email fallback) — once per route, no spam.
